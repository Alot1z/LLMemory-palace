# Task: Web Interface

**Scope:**
- `web/` directory with Express server
- `web/public/` - Static assets
- `web/routes/` - API routes

- `palace web` starts server on port 3000
- View palace status, files, patterns
- API endpoints: GET /api/status, /api/files, /api/patterns
- Basic HTML UI
